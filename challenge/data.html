MW-18Mar Dataset

This benchmark contains video sequences. We do not provide pre-computed detection. All formats and devkit for evaluation are consistent with MOT16 Challenge

Explain: FPS is Frames Per second. Resolution is Frame size (w x h). Length is number of frames (length in mm:ss).
Tracks is number of annotated trajectories. Boxes is Total number of annotated pedestrians.

Complete report on these videos will be released shortly.
